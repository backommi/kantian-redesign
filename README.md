# 瞰天科技官网 Redesign

🎨 三种不同视觉风格的企业官网

## 访问地址

**GitHub Pages：** https://backommi.github.io/kantian-redesign/

**三个风格版本：**
- 🚀 [风格 A - 科技未来](https://backommi.github.io/kantian-redesign/style-a/)：现代科技感、渐变配色、毛玻璃效果
- 🏢 [风格 B - 企业专业](https://backommi.github.io/kantian-redesign/style-b/)：深色主题、NVIDIA绿色点缀、企业级专业感
- 💼 [风格 C - 极简实验室](https://backommi.github.io/kantian-redesign/style-c/)：纯白背景、大字体留白、极简风格

## 技术栈

- HTML5 + Tailwind CSS（CDN）
- 响应式设计
- GitHub Pages 部署

## 快速本地预览

```bash
# 风格 A
open style-a/index.html

# 风格 B  
open style-b/index.html

# 风格 C
open style-c/index.html
```

## 项目结构

```
kantian-redesign/
├── index.html      # 入口页面
├── style-a/        # 科技未来风格
├── style-b/        # 企业专业风格
└── style-c/        # 极简实验室风格
```

## 部署说明

1. 推送到 GitHub 仓库
2. 在 Settings → Pages 中启用
3. 选择 master 分支
4. 等待 2-3 分钟自动部署
